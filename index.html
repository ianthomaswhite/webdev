Scoring Logic & Functions

Element Scoring Function

S = Q × [ (1 - P)(-D) + P(D + 0.5E) ]

  • Q (Multiplier): Scale factor (Standard = 1).
  • P (Presence): 1 if present, 0 if absent.
  • D (Desired): 1 (Yes), 0 (MB/Maybe), -1 (No).
  • E (Execution): Quality rating from -2 to +2.
